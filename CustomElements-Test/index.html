text_fields

Text

The text can be any type of different text elements, such as <p>, <h3>, <h6> and so on. The text is white (hex color code: #FAFAFA) and has a font-family value of "Roboto Slab". The font is fetched from Google Fonts.

Be careful not to write too much text on one slide since the overflowed text cannot be read!!

list

Lists

  • Lists look very similar to the normal text mentioned on the previous slide.
  • The only thing different about them is the bullet point.
  • An unordered list is not the only type of list available... there is also an ordered list!
  • Example:
    1. Bolded ordered list item
    2. A non-bolded orderd list item
    3. An anti-bolded list item (light font-weight)
 face

Icons

An icon can be used to explain, more visually, a slide's content. This project utilizes the Material Icons provided by Google Fonts. If you are interested in using this custom HTML element, then it is advised that you only use an icon at the beginning of a slide. All the previous slides are a good example of that.

not_interested

Not Supported

As amazing as this project is, it does not support all types of content. Reason being that the developer of this custom HTML element is simply just too lazy to do anything about it. Here is a list of all the HTML elements that do not have a solid support:

  • Images
  • Audio
  • Maps
  • SVG's
  • ...
